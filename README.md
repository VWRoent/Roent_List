# Roent.List（歌枠管理アプリ）

歌枠（カラオケ配信）向けに、曲の登録・検索・セットリスト管理・タイムスタンプ生成・OBS表示（HTML Viewer）までをまとめたローカルGUIアプリです。

- **プログラム名**: Roent.List  
- **バージョン**: 1.0.0  
- **制作日**: 2026年2月18日  
- **制作者**: VWRoent（紫波レント）  
- **使用技術**: ChatGPT  
- YouTube: https://www.youtube.com/@trans-cyp4365  
- Twitter(X): https://x.com/VioWaveRoentgen

## 主な機能

### 1. 曲データベース（登録）
- 曲名 / アーティスト / 音源提供元 / キーワード / 歌詞 / 概要欄記載事項
- 動画パス（mp4/mkv/webm）、音源パス（mp3/wav）
- 音源URL / 原曲URL など

データはローカルの **SQLite（songs.db）** に保存されます。

### 2. 検索タブ
- 曲名・アーティスト・音源提供元・登録キーワードを**部分一致**で検索
- 検索結果の一覧表示
- **詳細表示**、**セットリスト（キュー）追加**

### 3. 詳細タブ
- 曲情報を表示（曲名/アーティスト/音源提供元/キーワード/歌詞/概要欄記載事項）
- 曲名・アーティスト・音源提供元・キーワードは**クリックで同条件検索**
- 歌詞・概要欄記載事項は**クリックでクリップボードコピー**
- セットリスト追加 / 編集（登録タブへ引き継ぎ）にも対応

### 4. セットリストタブ
- 「現在歌っている曲」「キュー（Queue）」「歌い終わり（Done）」を管理
- キューの選択 → **Enter** で現在曲へ移動
- 現在曲の **音楽 / 動画 / 詳細 / BGM** ボタン
- 歌詞の表示・スクロール（「歌詞送り▼」「歌詞戻し▲」）
- タイマー（カウントアップ）と、曲開始時刻の記録（タイムスタンプ用）

### 5. スタンプタブ（YouTube用タイムスタンプ）
- 例：
  - `00:00 開始`
  - `mm:ss 曲名`
  - `hh:mm:ss 曲名`
- 1時間未満は `hh:` を省略
- 生成結果をクリップボードにコピー

### 6. OBS用 Viewer（HTML/CSS自動生成）
- `obs_viewer/view.html` と `obs_viewer/style.css` を自動生成
- **Queue と Done** を表示（配色・サイズ・フォント倍率などは設定タブから変更）

> 注: ViewerはローカルHTMLです。OBSのブラウザソースで「ローカルファイル」をONにして読み込みます。

### 7. 設定タブ
- ウィンドウテーマ（ピンク/オレンジ/ブルー/グリーン/ラベンダー/ダーク）
- OBS Viewer：全体サイズ（4:3 / 3:4 / 9:16 / カスタム）、文字倍率（1.5x / 1.0x / 0.8x）、配色、日時表示、Roent.List表示
- セットリスト：歌詞ボックスサイズ（小/中/大）
- BGM：動画/音源の参照、優先順位（デフォルト音源優先、チェックで動画優先）

## 動作環境

- Windows 10/11 推奨
- Python版: Python 3.10+（推奨 3.11+）
- EXE版: `roentlist.exe`（Python不要）

※GUIは tkinter を使用しています。通常は Python 同梱のため追加インストールは不要です。

---

## 使い方

### EXE版（Windows）
1. `roentlist.exe` をダブルクリックで起動
2. 初回起動時に、同じフォルダに以下が自動作成されます（存在しない場合）
   - `songs.db`（曲データベース）
   - `settings.json`（設定）
   - `obs_viewer/`（OBS Viewer一式）

### Python版
```bash
python roentlist.py
```

---

## OBSでの読み込み手順（Viewer）

1. OBSで「ソース追加」→「ブラウザ」
2. 「ローカルファイル」をON
3. `obs_viewer/view.html` を指定
4. 幅/高さを設定タブのViewerサイズに合わせる（例: 800×600）

---

## ファイル構成（目安）

```
roentlist.py            # Python版 本体
roentlist.exe           # Windows EXE版（配布物）
songs.db                # 曲DB（自動作成）
settings.json           # 設定（自動作成）
obs_viewer/
  view.html             # OBS Browser Source 用（自動作成）
  style.css             # Viewerスタイル（自動作成）
```
